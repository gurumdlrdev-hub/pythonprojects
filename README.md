# Multi-Purpose Python Automation Projects

This repository contains clean, user-interactive Python automation programs built for financial analysis and sports metrics calculations.

## Software Projects

### 1. State Fiscal Health Analyzer (economic.py)
This program analyzes the financial budget stability of a state using two main metrics:
* **Debt to GSDP Ratio:** Measures the total state debt against its total economic output to check if the debt level is safe, normal, or critical.
* **Interest to Revenue Ratio:** Compares annual interest expenditures against total revenue income to identify high-risk debt situations.

### 2. Cricket Metrics Analyzer (cricketcal.py)
This program calculates cricket performance metrics. It contains specialized functions to track:
* Batting Strike Rate
* Bowling Economy Rate
* Team Run Rate
* Player Performance Average

## Technical Stack and Code Features
* **Language:** Python 3.10
* **Input System:** Dynamic terminal-based user input handling.
* **Error Prevention:** Built-in exception handling to prevent crashes from invalid text inputs (ValueError) and dividing by zero (ZeroDivisionError).
* **Version Control:** Git and GitHub

## How to Run the Files

### Prerequisites
Make sure you have Python installed on your system.

### Execution
Open your terminal (PowerShell or Command Prompt) and run the following commands:

To test the budget analyzer:
python economic.py

To test the cricket analyzer:
python cricketcal.py
